Вывод текста

Можно выводить текст через return только в некоторых случаях, например в function(). В простом выводе лксше используйте output().

Например вместо "return ('hello')" используйте "output('hello')"

Установка расширения

1. Скопируйте URL или содержимое страницы https://shaman2016scratch.github.io/shaman2016.github.io/extensions/extension6/1.0.js

2. Откройте любой мод TurboWarp или сам TurboWarp.

3. Откройте библиотеку расширений

4. Загрузите пользовательское дополнение

5. Выберите нужное поле и вставьте скопированный код/url

6. Пользуйтесь!
