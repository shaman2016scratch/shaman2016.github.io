Установка расширения

Установить в Dashblocks

Вы можете установить библиотеку в Dashblocks через библиотеку расширений или через список версий тут

Установка в другую модификацию TurboWarp или в TurboWarp

1. Откройте главную страницу расширения

2. Выберите нужную версию

3. Скопируйте код версии или url адрес на код

4. Откройте библиотеку расширений

5. Загрузите пользовательское дополнение через код или url, выберите соответствующий способ загрузки и ведите url/код расширения

6. Загрузите без песочницы

Важно: версии 2.1+ поддерживаются только в Dashblocks
