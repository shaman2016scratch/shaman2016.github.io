RussiaScript

RussiaScript — новый русский язык программирования написанный на JavaScript.

Кто создал RussiaScript? RussiaScript создал русский 9-летний скретчер SHAMAN2016

Когда его создали? SHAMAN2016 создал RussiaScript 22.10.2025 (22 Октября 2025 года) в 23:38:54

Какой у него синтаксис? Синтаксис RussiaScript на половину русский, на половину английский

В каком формате RussiaScript? В формате Json, с ним легче всего взаимодействовать в JavaScript.

Сколько у него версий? У RussiaScript есть две версии, 1.0 и 1.1

Также SHAMAN2016 создал RSML (RussiaScript Markup Language) и SLRS (Style Lang RussiaScript)

Можно ли создавать в RussiaScript пользовательские функции? Да можно, для RussiaScript 1.1 можно создавать бибилиотеки на JavaScript а такде в RussiaScript 1.1 можно создавать пользовательские функции (на JavaScript или RussiaScript)

Также в RussiaScript есть внутренняя файловая система (filesRS) (она появилась в RussiaScript 1.1) с которой можно взаимодействовать через внутренний терминал (Терминал RussiaScript) который появился в RussiaScript 1.0

Для более удобного взаимодействия с filesRS можно сделать библиотеку для RussiaScript или же использовать пользовальские функции.

RussiaScript не экперементальный, не популярный.

Github проекта: https://github.com/shaman2016scratch/RussiaScript
