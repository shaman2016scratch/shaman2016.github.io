Что такое html?

Вступление

html — Расшифровывается как HyperText Markup Language (ру. Язык гипертекстовой разметки)

Он помогает создавать сайты*, а также он входит в web (html + css)

*Чтобы создавать сайты надо знать web (html + css) и JavaScript

Примение в web-разработке

с помощью html, css (web) браузеры понимают структуру сайтов (их внешний вид), в web

html показывает сам сайт, а css это стили, его можно встраивать в html с помощью атрибута

"style"

Встраивание

С помощью тега "iframe" можно встраивать сайты просто */*<*iframe src="*сайт*">*/* или же тоже самое но с дополнительными атрибутами.

Выполнение кода

С помощью тега "script" можно выполнять код JavaScript есть 2 вырианта выполнения:

1-ый

Написать код js внутри тега "script". Пример:

<*script">alert("hello world")
console.log("hello world")

2-ой

Добавить атрибут "scr" в котором будет ссыдка на код. Пример:

<*script src="shaman2016scratch.github.io/shaman2016.github.io/Курсы/1/1/js.js">
